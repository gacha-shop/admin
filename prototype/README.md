# 가챠맵 프로토타입 웹사이트 가이드

## 📁 파일 구조

```
가챠맵 프로토타입/
├── index.html          # 메인 랜딩 페이지
├── map.html           # 지도 페이지 (Phase 1)
├── community.html     # 커뮤니티 페이지 (Phase 2)
├── diary.html         # 가챠 다이어리 페이지 (Phase 2)
├── mypage.html        # 마이페이지 (Phase 3)
├── login.html         # 로그인 페이지
├── signup.html        # 회원가입 페이지
├── style.css          # 공통 스타일
├── map.css           # 지도 페이지 전용 스타일
├── community.css     # 커뮤니티 페이지 전용 스타일
├── diary.css         # 다이어리 페이지 전용 스타일
├── mypage.css        # 마이페이지 전용 스타일

```

## 🎨 페이지별 주요 기능

### 1. 메인 랜딩 페이지 (index.html)
**Phase: 공통**
- 히어로 섹션 (메인 배너)
- 서비스 소개 (4가지 핵심 기능)
- 인기 매장 TOP 5 미리보기
- CTA 섹션 (회원가입 유도)
- 푸터

### 2. 지도 페이지 (map.html)
**Phase 1: 코어 기능**
- 검색 바 (매장명, 지역, 상품 검색)
- 필터 패널 (매장 타입, 결제 방식, 영업 상태)
- 지도 영역 (마커 표시)
- 매장 리스트 (거리순, 별점순 정렬)
- 매장 제보 플로팅 버튼
- 실시간 영업 상태 표시
- 관심 매장 하트 버튼

### 3. 커뮤니티 페이지 (community.html)
**Phase 2: 커뮤니티**
- 탭 메뉴 (전리품 인증샷, 리뷰, 오늘의 럭키샵)
- 게시물 피드 (인증샷, 리뷰)
- 좋아요, 댓글, 북마크 기능
- 해시태그 시스템
- 오늘의 럭키샵 위젯
- 인기 해시태그
- 이번 주 베스트 유저 랭킹

### 4. 가챠 다이어리 페이지 (diary.html)
**Phase 2: 커뮤니티**
- 통계 대시보드 (총 뽑기, 지출, 방문 매장)
- 필터링 (기간, 매장, 정렬)
- 월별 다이어리 엔트리
- 월별 지출 그래프
- 인기 시리즈 랭킹
- 자주 가는 매장
- 위시리스트

### 5. 마이페이지 (mypage.html)
**Phase 3: 재미 요소**
- 프로필 헤더 (아바타, 통계)
- 탭 메뉴 (내 활동, 뱃지&스탬프, 관심 매장, 설정)
- 작성한 리뷰 목록
- 올린 인증샷 그리드
- 제보한 매장 (승인 상태)
- 획득한 뱃지 (8/20)
- 방문 스탬프 진행도
- 관심 매장 목록

### 6. 로그인/회원가입 (login.html, signup.html)
**Phase 1: 코어 기능**
- 이메일/비밀번호 로그인
- 소셜 로그인 (카카오, 구글, 네이버)
- 랜덤 닉네임 생성 버튼
- 이용약관 동의

## 🎨 디자인 특징

### 컬러 팔레트
- Primary: #6366f1 (보라)
- Secondary: #8b5cf6 (퍼플)
- Success: #10b981 (초록)
- Warning: #f59e0b (주황)
- Danger: #ef4444 (빨강)

### 주요 UI 요소
- 그라데이션 배경 (보라-퍼플)
- 부드러운 그림자 효과
- 12px 둥근 모서리
- 부드러운 호버 애니메이션
- 반응형 디자인 (모바일 최적화)

## 💡 주요 인터랙션

### 호버 효과
- 카드: 위로 살짝 올라감 + 그림자 강화
- 버튼: 색상 변화 + 위로 살짝 올라감
- 링크: 색상 변화

### 애니메이션
- 지도 마커: 위아래 바운스 애니메이션
- 하트 버튼: 클릭 시 하트비트 애니메이션
- 로딩 상태: 페이드 인/아웃

## 📱 반응형 디자인

### 데스크톱 (1024px 이상)
- 2단 레이아웃 (메인 + 사이드바)
- 풀 네비게이션 메뉴
- 그리드 3-4열

### 태블릿 (768px - 1024px)
- 1단 레이아웃
- 간소화된 네비게이션
- 그리드 2열

### 모바일 (768px 이하)
- 1단 레이아웃
- 햄버거 메뉴 (미구현)
- 그리드 1열
- 스와이프 가능한 탭

## 🚀 사용 방법

1. **로컬에서 실행**
   ```bash
   # 간단히 index.html을 브라우저에서 열기
   open index.html
   
   # 또는 Python 서버 실행
   python -m http.server 8000
   # http://localhost:8000 접속
   ```

2. **페이지 네비게이션**
   - 메인 → 지도/커뮤니티/다이어리
   - 헤더의 로고 클릭 → 메인으로 이동
   - 각 페이지 간 링크로 이동

3. **프로토타입 테스트**
   - 모든 버튼과 링크는 시각적으로만 작동
   - 실제 기능은 구현되지 않음 (백엔드 필요)
   - 반응형 디자인 확인을 위해 브라우저 크기 조절
